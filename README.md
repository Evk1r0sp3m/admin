# Admin

This repository contains everything you need to know about GoSheffield.

### Summary

GoSheffield is a monthly meetup in Sheffield about the [Go Programming
Language](https://golang.org/). Organised by the local community, we focus on
short technical talks and fostering the growth of Go in the area.
